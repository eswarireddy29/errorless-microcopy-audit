# Errorless — Microcopy Audit & Redesign

## Overview
End-to-end microcopy audit of a 7-step e-commerce checkout 
flow, focused on reducing friction at error moments.

## Problem
Users were dropping off at the payment step due to unclear, 
technical error messages and vague form validation copy.

## My Role
UX Writer — audited all checkout copy, rewrote error messages 
using plain language, built a reusable Error Message Playbook.

## What's Inside
- Full audit of 50+ strings across 7 checkout steps
- Before & after copy comparisons
- Error Message Playbook (30+ templates)
- Usability testing results

## Key Results
- 42% drop in payment-step abandonment after copy changes
- Error Message Playbook adopted as team reference document
